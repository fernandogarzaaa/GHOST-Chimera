# Submission Kit (All Tracks)

## Product name

**Ghost Chimera — Governed Enterprise Change Agent**

## Short description

Ghost Chimera is a local-first governed AI change agent that converts repository context into policy-checked plans, auditable execution, and PR-ready outputs.

## Long description

Ghost Chimera helps teams modernize software safely by combining orchestration, policy enforcement, and replayable telemetry. It compiles objectives into task specs, selects runtime backends with bounded autonomy, verifies outputs, and records evidence for trust and review.

## Suggested tags

`agentic-ai`, `enterprise-ai`, `governance`, `security`, `devtools`, `orchestration`, `automation`, `software-engineering`, `local-first`

## Architecture one-pager bullets

- Task IR + compiler (`TaskSpec` / `TaskKind`)
- Backend scheduler with fallback and bounded parallel
- Policy gates for network/python/shell/desktop and production guardrails
- Verification and confidence envelope
- Replayable telemetry and audit outputs
- Operator console for status, jobs, schedules, and readiness

## Judging-mapped value bullets

- **Innovation:** unified governance + execution for repo change workflows
- **Technical depth:** multi-backend scheduling, policy traces, runtime verification
- **Practicality:** reproducible CLI + console + deployable container
- **Trust:** explicit boundaries, production gating, audit-ready outputs
