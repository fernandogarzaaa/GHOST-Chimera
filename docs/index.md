# Ghost Chimera Documentation

Ghost Chimera is a local-first agent orchestration platform for building role-specific Ghost operators with safety guardrails, provider routing, memory surfaces, and developer automation.

Start with the user tutorial if you are new to the product. Use the quick start guide when you want the shortest install path, then review the deeper operational and developer docs.
