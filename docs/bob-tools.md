# IBM Bob Tools

Bob tooling provides local developer acceleration without network calls or secrets.

## Reports

- `scripts/bob_accelerator.py`: repository readiness and onboarding report.
- `scripts/coverage_report.py`: source-to-test coverage visibility.
- `scripts/bob_delivery_package.py`: PR-ready delivery package.

## Developer Utilities

- `scripts/generate_changelog.py`: changelog draft from git history.
- `scripts/validate_config.py`: production guardrail validation.
- `scripts/audit_dependencies.py`: dependency specification audit.
- `scripts/generate_test_scaffold.py`: AST-based pytest scaffold generation.
- `scripts/generate_api_reference.py`: AST-based API reference generation.
- `scripts/generate_sbom.py`: SBOM-lite generation.
- `scripts/dependency_graph.py`: internal import graph.
- `scripts/analyze_logs.py`: log summary and repeated-message detection.
- `scripts/dev_env.py`: setup command profiles.
