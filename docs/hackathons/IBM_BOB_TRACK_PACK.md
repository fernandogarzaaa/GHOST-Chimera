# IBM Bob Track Pack

## Purpose

Package Ghost Chimera for the IBM Bob Hackathon requirement: AI as a codebase-aware dev partner that improves software development workflows.

## Bob + Ghost operating boundary

- **IBM Bob does:** repository-aware analysis, issue/backlog drafting, improvement suggestions, and report export.
- **Ghost Chimera does:** governed orchestration, policy-gated execution path, confidence/verification framing, and audit-ready package creation.

## Recommended workflow

1. Run Bob analysis on repo and export report.
2. Place exported report in `docs/hackathons/artifacts/`.
3. Convert report actions into Ghost Chimera objective(s).
4. Execute governed flow and produce output package.
5. Submit both artifacts together: Bob report + Ghost governed package.
