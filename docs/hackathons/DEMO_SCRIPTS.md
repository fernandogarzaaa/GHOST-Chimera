# Demo Scripts (One Backend Flow, Four Lenses)

## Shared demo skeleton

1. Load repo context and objective.
2. Show governed plan output.
3. Show policy/security checks and confidence.
4. Show PR-ready package and audit trail.

## 1) Agentic Olympics lens

Focus: autonomous coordination with bounded delegation.

- Emphasize autonomy profiles, strategy ceilings, and fallback.
- Show that delegation remains policy-scoped and auditable.

## 2) Transforming Enterprise Through AI lens

Focus: security and explainability.

- Emphasize production guardrails, approval pathways, and policy traces.
- Show confidence + provenance output and readiness checks.

## 3) IBM Bob Hackathon lens

Focus: AI dev partner improving software delivery.

- Use Bob for repository-level analysis and backlog suggestions.
- Feed that into Ghost Chimera to produce governed execution and output package.
- Export and attach Bob artifact (see IBM Bob track pack).

## 4) AI GENESIS lens

Focus: scalable and adapter-ready architecture.

- Emphasize modular backends, plugin surfaces, and deployment posture.
- Show same workflow can pivot to enterprise/partner contexts without re-architecture.
